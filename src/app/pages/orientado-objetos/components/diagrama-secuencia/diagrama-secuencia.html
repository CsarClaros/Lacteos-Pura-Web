Diagrama de Secuencia

El diagrama de secuencia muestra la interacción temporal entre los diferentes objetos del sistema durante el proceso de registro de un pedido, destacando el orden de los mensajes intercambiados.

Caso de Uso: Registrar Pedido

Elementos del Diagrama

  • Líneas de vida: representan la existencia de los objetos.
  • Mensajes: llamadas entre objetos.
  • Retornos: respuestas generadas por los objetos.
  • Secuencia temporal: el tiempo avanza de arriba hacia abajo.

Descripción del Flujo

El cliente inicia sesión en el sistema y selecciona los productos deseados. El sistema consulta el inventario para verificar la disponibilidad de stock y posteriormente genera el pedido.

Finalmente, el pedido se almacena en la base de datos, se actualiza el inventario y se muestra la confirmación al usuario.
