Modelo Ambiental

El modelo ambiental define el contexto del sistema, identificando las entidades externas que interactúan con Lácteos Pura y los flujos de información principales. Este modelo establece los límites del sistema y su relación con el entorno empresarial.

Actores Externos Identificados

  • Proveedores de materia prima
  • Clientes mayoristas y minoristas
  • Personal administrativo y operativo
  • Distribuidores y transportistas
  • Entidades regulatorias (SENASAG)

Diagrama Entidad-Relación

Modelo de Datos del Sistema

El Diagrama Entidad-Relación representa la estructura de la base de datos, mostrando las entidades principales del negocio, sus atributos y las relaciones entre ellas.

Entidades Principales

  • PRODUCTO: Artículos lácteos comercializados
  • CLIENTE: Compradores mayoristas y minoristas
  • PEDIDO: Solicitudes de compra
  • INVENTARIO: Control de stock
  • PRODUCCION: Registro de manufactura

Cardinalidades

  • Un cliente puede tener muchos pedidos
  • Un pedido tiene muchos detalles
  • Un producto tiene un registro de inventario
  • Un producto puede tener muchas producciones
