Análisis de Sistemas

Definición

El análisis de sistemas consiste en estudiar una organización, identificar sus procesos, problemas y oportunidades de mejora para diseñar soluciones tecnológicas eficientes.

Fases del Análisis

1. Planificación

Definición de objetivos y alcance.

2. Análisis

Levantamiento de requerimientos.

3. Diseño

Modelado de la solución.

4. Implementación

Construcción y puesta en marcha.

Modelo Estructurado

El análisis estructurado se enfoca en los procesos y flujos de datos.

Diagrama de Flujo de Datos (DFD)

Representa cómo circula la información dentro del sistema.

Diccionario de Datos

Describe cada dato utilizado por el sistema.

Diagrama Entidad Relación

Define las entidades y relaciones de la base de datos.

Modelo Orientado a Objetos

Modela el sistema mediante objetos que contienen atributos y métodos.

Clase

Plantilla que define objetos.

Objeto

Instancia de una clase.

Herencia

Permite reutilizar características.

Polimorfismo

Diferentes respuestas ante el mismo mensaje.
